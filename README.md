# SATYA — Synthetic Audio & Video Authenticity
## सत्य | Where Truth Speaks Every Language

> **Team:** Imperials | **Member:** Sanjay Ravichandran | **Event:** AWS AI for Bharat Hackathon 2026

SATYA is a multi-platform, multi-lingual AI content authenticity verification system that detects deepfake videos, synthetic audio, and AI-generated text across 11 Indian languages and 4 major social platforms.

### 📄 Documentation
- [requirements.md](requirements.md) — Product Requirements Document
- [design.md](design.md) — Technical Design Specification

### 🛡️ Key Features
- 🎭 Multi-modal AI detection (video + audio + text)
- 🌐 11 Indian language support
- 📊 SATYA Score (0–100) with explainable evidence
- 📱 YouTube, Instagram, ShareChat, X integration
- 🗺️ India Trust Heatmap dashboard

### 🏗️ Tech Stack
Python (FastAPI) · React · AWS (ECS, Lambda, S3, DynamoDB, Bedrock, Translate) · ONNX Runtime · OpenAI Whisper · FFmpeg
